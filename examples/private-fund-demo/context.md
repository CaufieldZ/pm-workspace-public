## 2. 产品现状 `静态`

财富管理 App 当前 v1.0 已支持：

- **基金列表**：H5 展示在售私募基金，按风险等级筛选
- **基金详情**：基础信息、净值走势、历史业绩、产品说明书下载
- **合格投资者认证**：身份认证 + 资产证明上传 + 风险测评（C1-C5 风险等级）
- **运营后台**：基金录入、净值发布、投资者审核

**v1.0 未覆盖**：认购下单、申购追投、赎回流程、订单状态流转、电子签约、冷静期管理。
本项目从 v1.1 起补齐认申赎主链路，让合格投资者在 H5 内完成从认购到赎回的全流程。

## 6. 业务规则 `静态`

### 合规规则（监管要求，不可破）

- **R1 合格投资者门槛**：金融资产 ≥ 300 万元，或近 3 年年均收入 ≥ 50 万元
- **R2 风险匹配**：投资者风险评级 ≥ 基金风险等级（C3 投资者只能买 R1-R3 基金）
- **R3 单笔起投**：≥ 100 万元
- **R4 冷静期**：合同签署后 24 小时内可单方解除，资金原路退回，不收取费用
- **R5 双录**：销售过程需录音录像（H5 用电子签 + 视频确认替代）

### 业务规则

- **B1 开放日**：每月最后一个工作日为申购/赎回开放日
- **B2 净值披露**：每个开放日 17:00 前发布最新净值
- **B3 申购确认**：T+1 工作日确认份额（T = 开放日）
- **B4 赎回到账**：T+5 工作日资金到账（T = 开放日）
- **B5 大额赎回顺延**：单次赎回 > 总份额 10% 的部分顺延至下一开放日，按下一开放日净值结算
- **B6 订单状态机**：
  - 认购：待签约 → 待支付 → 募集中 → 成立确认 → 已成立 / 募集失败
  - 赎回：已申请 → 风控审核 → 待结算 → 已到账（大额可能进入「部分顺延」状态）
